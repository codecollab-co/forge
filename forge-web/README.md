# forge-web

TypeScript + Next.js. Product UI and marketing site.

Talks to `forge-platform` for state, to `forge-agent` for live Run streams (SSE) only. No business logic in the BFF — thin proxy where needed.

See [`../ARCHITECTURE.md`](../ARCHITECTURE.md) and the ADRs at [`../docs/adr/`](../docs/adr/).

## Status

Pre-implementation. Scaffolding lands as part of issue #1 (walking skeleton).
